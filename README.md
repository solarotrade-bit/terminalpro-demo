# TerminalPro - Modern Bloomberg Terminal Clone

## Demo Funcional

Este es un demo funcional del TerminalPro, un clon moderno y asequible del Bloomberg Terminal con características avanzadas de IA y análisis de datos alternativos.

### Características Demostradas

- **UI/UX Moderna**: Interfaz limpia y profesional con temas oscuros
- **Streaming de Datos en Tiempo Real**: Actualizaciones de precios cada 3 segundos
- **Gráficos Interactivos**: Charts de velas japonesas con múltiples timeframes
- **Watchlist**: Lista de seguimiento con precios actualizados
- **Análisis Técnico**: Indicadores como P/E Ratio, Market Cap, Volume
- **Noticias en Tiempo Real**: Feed de noticias con análisis de sentimiento
- **Búsqueda Rápida**: Sistema de búsqueda y comandos

### Tecnologías Utilizadas

- **Frontend**: HTML5, CSS3, JavaScript
- **Gráficos**: Lightweight Charts (Bloomberg-like)
- **Datos**: Simulación de datos reales de mercado
- **Diseño**: Responsive design para todas las pantallas

### Cómo Probar

1. Abre el archivo `index.html` en tu navegador
2. Observa cómo los precios se actualizan en tiempo real
3. Haz clic en diferentes símbolos en la watchlist
4. Cambia entre diferentes timeframes
5. Explora las diferentes secciones del terminal
